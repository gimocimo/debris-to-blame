# handoffs/

Per-session logs — the project's memory between sessions. Anti-drift discipline mirrored from
`learning-to-defer-llm`.

- **Start of session:** read `PROJECT_PLAN.md §1` (Current Status) + the **latest** handoff here.
- **End of session:** copy `TEMPLATE.md` to `NNNN-short-slug.md`, fill it, then update
  `PROJECT_PLAN.md §1` and the §8 traceability matrix if evidence changed.
- One file per session, zero-padded, append-only. Never edit an old handoff except to fix typos.
